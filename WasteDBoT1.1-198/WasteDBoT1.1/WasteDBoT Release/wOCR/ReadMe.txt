How to use WasteDBoT

1) Auth in WasteDBoT

2) Load wOCR.exe 

3) Click AUTH!

4) Enter your details (CASE Sensitive) Press ok
You should get Auth Successful, If you dont retry. If it still fails please email us with
your details.

5) Once Auth'ed Click hold down the Black Square Box and drop in the RS APPLET not any part
of the screen. You MUST drop it in the applet (It will be outlined in red to help determine this.)

7) If you wanna turn it off then press off!

8) Should you need multiple client support this is coming in the next version so for the moment
run 2 clients.
