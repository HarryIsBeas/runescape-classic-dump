RuneScape Advanced v1.0 by RANCID

Features

Here is a list of the key features in RuneScape Advanced:
    • *new* Automatically load any script placed in START folder when RSAdvanced is started
    • *new* Start RSAdvanced with windows
    • *new* Sleak, small, and fast Interface
    • *new* Custom Scripts Support in the BASIC language. Automate what you want to do, how you want to do it.
    • *new* Scripts have the ability to specify Author Name and Script title, which will be displayed when the script is loaded
    • *new* Now fully supports Members Servers
    • *new* Read the RS news from RSAdvanced fast and easy
    • *new* Quickly get High scores just by entering the players name
    • Save a Screenshot quick and easy (File> Save Screenshot)
    • Automatically load a certain world when program starts
    • Automatically log back in when logged out
    • Get the world player counts quickly when you need them
