What is PiPe?
As you are aware from PiPe v1 it is a tunnel aka port redirector.

How do i use?
Launch PiPe.exe and select the IP + port of the server your connecting to. You may choose ur own
also edit the local port. Click the servers tab for more help.

Once selected simply press start and fire away!
