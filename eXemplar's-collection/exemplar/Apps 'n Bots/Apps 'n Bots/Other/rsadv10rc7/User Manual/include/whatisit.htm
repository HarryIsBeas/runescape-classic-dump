What is RuneScape Advanced?

Runescape Advanced is a complete self contained RuneScape.com client program with a very powerful built in scripting ability.
    Key features include:
      • Powerful and easy to use scripting engine allows you to do basicly whatever you want
      • Fast; Advert free, client loader
      • Easy to use User Interface
      • 9x/Me Fully Supported (unlike SCAR)
      • Tons of Scripts available
Will I get banned?
RS Advanced (RuneScape Advanced) loads the RuneScape client directly from RuneScape.com in a fast and clean method. There is no danger of being detected as in other Automaters such as RuneBot (originally by Vee) or other modified RuneScape clients, because as far as Jagex™ can tell your playing right off RuneScape.com. The only way you could get discovered is by someone reporting you, so dont be stupid and auto in full worlds in a popular place. Best place to auto is in a remote location on a near empty world.
