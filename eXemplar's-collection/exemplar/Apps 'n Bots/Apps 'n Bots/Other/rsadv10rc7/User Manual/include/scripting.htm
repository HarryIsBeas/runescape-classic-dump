RuneScape Advanced v1.0 by RANCID

Scripting Reference

In RS Advanced you can easily create and debug your custom scripts through the Script Editor. Continue reading on to learn more about the commands available and syntax (the programming language) used to create scripts.

    Commands

    Functions: Description: Example:
    FindColorEx(color As Long, Optional XStart = 0, Optional YStart = 0, Optional Xwidth = 511, Optional YWidth = 344, Optional Steps = 1) As Long Scans the specified portion of the window for the specified color. lngPos = FindColorEx(0, 353, 73, 155, 115)
    FindColor(color As Long, Optional Steps = 1) As Long Scans the entire window for a color and returns the LONG coordinates. If color is not found returns -1. Steps is the number of pixels to scan (ie; every 5) lngPos = FindColor(vbRed)
    GetX(lngCoords as long) as Integer Returns the X coordinate from a LONG value. GetX(7864520)
    GetY(lngCoords as long) as Integer Returns the Y coordinate from a LONG value. GetY(7864520)
    SetFightMode(Mode As Integer) Sets the current fight mode [if fighting]. SetFightMode(1)
    ChangeWorld(World As Integer) Loads/changes to the specified world in the client window ChangeWorld(10)
    ClickRune(X As Integer, Y As Integer) Sends a left click message at the specified coordinates ClickRune(120, 200)
    rClickRune(X As Integer, Y As Integer) Sends a right click message at the specified coordinates rClickRune(120, 200)
    lClickRune(lngPos as Long) Sends a click message at the specified LONG type coordinate [this is a intermediate command, if you don't understand it do not try to use it] lClickRune(7864520)
    MoveRune(X As Integer, Y As Integer) Sends a mouse move message to the specified coordinates MoveRune(120, 200)
    lMoveRune(lngPos as Long) Sends a move message at the specified LONG type coordinate [this is a intermediate command, if you don't understand it do not try to use it] lMoveRune(7864520)
    TypeRune(strMessage as string) Types out the specified message. Note; the message will send in lowercase and will not display "!@#$%^&*()_+|{}:"<>?" unless the user is holding shift down TypeRune("hello")
    DebugMessage(strMessage As String) Displays the message in the debug window in Script Editor DebugMessage("Logged in? " & LoggedIn)
    MoveToColor(Xstart As Long, YStart As Long, color1 As Long, Optional color2 As Long, Optional color3 As Long, Optional color4 As Long, Optional Secs2Scan As Integer) As Long Scans the window until the color is found (or time limit hit), then sends MoveRune command to it. Xstart and Ystart are and optional way to specify where to start the search from. Secs2Scan is the amount of time to take scanning in seconds. MoveToColor also returns the Long position of the coords lResult = MoveToColor(0, 0, 733305, 533335, 1132725,,2)
    CompareImage(imageFile As String, x As Integer, y As Integer, Optional transColor As Long) As Boolean Compares the image with the current pixels in the window at the x and y coordinates and returns True or False. TransColor is if you would like to specify a transparent background for the image your comparing. CompareImage("c:\images\accessbank.bmp", 6, 5, vbMagenta)
    StatusText(strStatus As String, Color As Long, Optional x = 6, Optional y = 5) As Boolean Compares if the current Status Text (Example: "Guard (Level-27)") is the same string. StatusText("Guard", vbYellow)
    Wait(interval) Waits the specified amount of seconds before continuing in the script Wait(1.5)
    GetColor(X, Y) As Long Returns the color found at the specified X and Y coordinates lColor = GetColor(12, 12)
    msgInput(strMessage As String, strDefault As String) Prompts the user of the script for input strMessage is the question/message to prompt them with strDefault is the default input. strUserName = msgInput("Type your username", "[type here]")
    LockWindow(bLock As Boolean) Locks the RuneScape window; allowing no input from the user. Use this to allow the user to hover around, and in the rs window, without interupting the script. LockWindow(True)
    SaveScreenShot() Writes a screenshot to file SaveScreenShot()
    FindObject(strObject As String, objCol As Long, SearchColor As Long, Optional xStart = 0, Optional yStart = 0, Optional xWidth = 511, Optional YWidth = 328) This will search the window for the specified color, then compare the StatusText - if its true the function will end and return the LONG coordinate, if it is false it will continure searching until it has reached the end of the search range. Returns -1 if the Object was not Found. lResult = FindObject("Ladder", 16776960, 9547)

    Properties

    Property: Description:
    Fighting As Boolean Returns if you are currently Fighting.
    IsSleeping As Boolean Returns if you are currently Sleeping.
    WelcomeScreen As Boolean Returns if the f2p welcome screen [black box on login] is currently displayed or not.
    LoggedIn As Boolean Returns if you are currently at the f2p login screen.
    p2pLoggedIn As Boolean Returns if you are currently at the p2p login screen.
    p2pLoggedIn As Boolean Returns if you are currently at the p2p login screen.
    LoginScreen As Boolean Returns if you are currently at the p2p/f2p login credential screen [ie; UserName and Password screen].
    CurrentWorld as Integer Returns the currently loaded RS World.
    UserName as String Returns the UserName specified in the options dialog [note; this may be removed in future versions].
    Password as String Returns the Password specified in the options dialog [note; this may be removed in future versions].
