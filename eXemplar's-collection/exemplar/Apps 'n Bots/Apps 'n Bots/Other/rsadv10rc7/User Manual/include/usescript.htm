Using a Script

RuneScape Advanced makes it easy to load customs scripts:
    Any scripts you may download/create should be placed in the Scripts folder in the RuneScape Advanced directory. When loaded, RuneScape Advanced will load all scripts in the folder into the scripts menu, simple select the script you wish to run from this memory to begin it (see example to the right).

    There are also other ways to load scripts;
    1.) quickly open RuneScape advanced by double clicking on a script file to launch it
    2.) Open a script inside the Script Editor to debug or view a script
