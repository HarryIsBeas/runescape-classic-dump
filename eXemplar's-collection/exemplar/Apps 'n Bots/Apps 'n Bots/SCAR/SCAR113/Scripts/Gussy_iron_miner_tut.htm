Chapter 2 – Colors

 

Getting the right colors is obviously important.

 

Firstly

 

Primary Importance colors (must take)

 

Inventory

 

Invhead, take the color you see most of, and make sure isn’t the dark color that is the handle (that will screw the script over)

 

Handle, the dark brown of the pick handle, should be a four digit color

 

Minimap

 

Mining spot, take the color that is the outer circle of the mining spot (see here if you don’t understand)

 

Banklogo, the brightest gold color

 

Road, the road color (minimap) that is next to the east bank, on the west side

 

Screen

 

Rock1, this is simple take a color of the rock (iron)

Rock2, as above

Rock3, as above

Rock4, as above

 

The purpose of the four colors is to randomise the colors clicked on, for anti banning reasons, the colors you have selected are used in a random order

If you are lazy you can use the same color or only two or three colors, but four is best for anti ban reasons.

 

Head, this is the color of the pickaxe head on the screen, so that the script can find it when it comes off.  This is an important color, try wielding the pickaxe to get the color, but make sure you unwield it before running the script.

 

Secondary Importance Colors

 

These colors either do not need to be changed or are only used if using the rimmington protection.

 

Firstly

The non change colors.

Both invore and bankbooth generally work fine as they are, however if you run into trouble with the script, try retaking these colors.

 

If the script continues to mine when inventory full, firstly make sure that the inventory is open, if problem persists take a new invore.  This is just the dark brown centre bit of the ore.

 

If the banking doesn’t click on the bankbooth (eventually)

Re take the bankbooth, this is the brown wood trim of the bankbooth.

 

Secondly

The rimmington colors

 

These colors are used as part of the anti get lost, which will check if it is in rimmington, I am not going to explain how it works, but if you wish to use rimmington protection, which I recommend, take these colors.

 

Magic, this should be the brightest orange (or close to) on the magic shop logo in the minimap

Water, this should be the most vibrant blue, which is near the bottom of the logo on the minimap.

 

That is all the colors, if you have an other questions pm me or post a reply.

 

Chapter 3 – Forms

 

The form in the script is for ease of use with the many selectable options.

 

(see picture here)

 

First, choose random mode allows you to select the type of random protection, I suggest waamp, but if you do not have waamp run the in script one(atons), or if you are using your own random event, select none.

 

Secondly, use rimmington correction, this procedure detects when the walking procedure has malfunctioned and will correct it.

 

Thirdly, Use gussy’s pick fix, this is not very good, I suggest using the waamp one, in which case don’t tick this box.

 

Fourthly, use mine timer.  The mine timer is a device that determines the time that the script has taken between mining ores.  This is especially useful if you end up in lumby and there are similar colors, which the script clicks on.  It basically prevents stupid malfunctions by then logging out.  Tick this box to use it, I recommend using it.  Underneath this is a drop list; this is a selection of times between ores.  I personally use five minutes, but feel free to use whatever you want.

 

Next, the walk timer works in the same way as the mine timer by checking if the walking procedure has taken too long to get to the bank or to the rocks.  Again I use 5 minutes.

 

Nearly done, the use auto talk will allow the script to . . .use auto talking (see chapter 4)

 

Finally username and password, I’m not going to explain these, that’s obvious.

 

Please note: the script tells you what it has registered as you selecting in terms of features in the message area, check this.

This feature wasn’t going to originally be included in the script due to its temperamental state, however I have made a selectable option to use it.  To use the auto talking check the use auto talk in the main form.

 

The auto talk will randomly say one of your entered phrases any time from 5 minutes after you begin mining.
