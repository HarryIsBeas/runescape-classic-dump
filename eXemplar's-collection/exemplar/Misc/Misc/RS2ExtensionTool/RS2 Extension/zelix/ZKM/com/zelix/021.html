Overview

Use this dialog to choose the type of parameter that you wish to add to the Trim Exclusion window's "Exclusion parameter" list.

Select the "Class" radio button if you wish to add an exclusion parameter that will prevent a class from being removed.

Similarly, select the "Field" radio button if you wish to add an exclusion parameter that will prevent a field from being removed.

Finally, select the "Method" radio button if you wish to add an exclusion parameter that will prevent a method from being removed.
