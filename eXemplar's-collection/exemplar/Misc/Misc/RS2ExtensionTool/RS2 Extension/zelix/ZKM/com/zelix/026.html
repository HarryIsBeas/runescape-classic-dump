Overview

Use this dialog to choose the type of parameter that you wish to add to the Obfuscate Name Exclusion window's "Exclusion parameter" list.

Select the "Package" radio button if you wish to add an exclusion parameter that will prevent a package from being renamed.

Select the "Class" radio button if you wish to add an exclusion parameter that will prevent a class from being renamed.

Select the "Field" radio button if you wish to add an exclusion parameter that will prevent a field from being renamed.

Similarly, select the "Method" radio button if you wish to add an exclusion parameter that will prevent a method from being renamed.

Finally, select the "Class suffix" radio button if you wish to add an exclusion parameter that will preserve a specified class name suffix and "link" the renaming of classes with that suffix to that of classes with names matching the non-suffix part of the name.
