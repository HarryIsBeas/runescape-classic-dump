Overview

This window displays the ZKM Script produced by the "ZKM Script Helper" or the "Build Helper". Click the "Save" button to save the script to disk. A ZKM Script can be run from the command line by using the "java -jar ZKM.jar myScript.txt" syntax.

It is highly recommended that, once you have become comfortable with the basics of Zelix KlassMasterTM and trimming/obfuscation, you use the ZKM Script interface rather than the Zelix KlassMaster GUI for your routine trim and obfuscation tasks. ZKM Scripts allow you to automate the task and to use more complicted features. You can edit a ZKM Script using any text editor.
