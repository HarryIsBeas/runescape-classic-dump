Overview

Use this dialog to specify which classes will be opened for processing within Zelix KlassMasterTM. You can specify
  • folders,
  • folders and all their subfolders,
  • archive files with the following extensions
    • jar
    • zip
    • war
    • ear
All the class files in the selected folders and archives will be processed. It is highly recommended that you process your application as a whole. If you do not do this then it could cause problems for Zelix KlassMaster's "name obfuscation" and "trim" functions.

File selection list

Select the directories and archives that contain the classes that you wish to process then click the ">>" button to copy them to the "Selected Folders and Archives" list.

">" button

Click this button to copy selected folders and archives from the "File selection list" to the "Selected Folders and Archives" list.

">>" button

Click this button to copy selected folders, the subfolders of selected folders and selected archives from the "File selection list" to the "Selected Folders and Archives" list.

"<" button

Click this button to remove the selected items in the "Selected Folders and Archives" list.

"<<" button

Click this button to remove all the items in the "Selected Folders and Archives" list.

"Selected Folders and Archives" list

The list of folders and archives whose class files will be processed.
