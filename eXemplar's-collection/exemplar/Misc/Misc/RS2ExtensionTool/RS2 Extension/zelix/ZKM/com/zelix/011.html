ZKM Script Helper

The Script Helper will lead you through the steps required to create a basic ZKM Script file. The Script Helper should be used in preference to the Build Helper for larger applications or complex applications. These are more conducive to errors during the online execution of steps by the Build Helper.

Once the Script Helper has completed its task of generating and saving the ZKM Script file, you can enhance the script file using any text editor, and then execute the script file from the command line.

The ZKM Script Helper will help you create a basic ZKM Script file by leading you through a sequence of dialog screens. These screens will allow you to build a ZKM Script that will

  • set the classpath that Zelix KlassMasterTM should use,
  • open your classes,
  • set your trim exclusions,
  • trim your classes,
  • set your name obfuscation exclusions,
  • obfuscate your classes,
  • save your trimmed and/or obfuscated classes.

You can exit the sequence at any time by pressing the "Cancel" button. You can return to an earlier step by pressing the "Previous" button.
